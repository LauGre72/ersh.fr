# Kit web ERSH.fr

Contenu :
- `logo-complet.png` : logo principal, fond transparent.
- `logo-carre.png` / `logo-carre-512.png` : version carrée.
- `logo-fond-bleu.png` : version sur fond coloré.
- `favicon.ico` + `favicon-*.png` : favicons multi-tailles.
- `palette.json` : couleurs de la charte.
- `ersh-theme.css` : variables CSS.
- `site.webmanifest` : manifeste web app.
- `head-snippet.html` : balises HTML à intégrer dans le `<head>`.

Note : les fichiers sont extraits depuis l’image fournie. Pour une qualité parfaite en très grand format, il faudra repartir d’un fichier vectoriel source.
